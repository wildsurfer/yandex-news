== Description ==

Yandex.News plugin generates an XML feed, called YML, for export of posts in the Yandex.News service.

Features:
* the possibility to specify the path to the feed
* filter categories

Localization

* Ukrainian (uk_UA)
* Russian (ru_RU)

Плагин «Яндекс Новости» предназначен для генерации XML-потока, называемого YML, используемого в качестве источника одноименным сервисом.

Особенности:
* возможность самостоятельно задать путь к потоку
* фильтр категорий

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `yandex-news` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in the 'Settings' admin meny.
